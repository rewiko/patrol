
PATROL
------

To run a local demo with a couple bots fighting against each others:

1) run it.unipr.ce.dsg.patrol.GUI.RTSGameBootStrapServer

2) run it.unipr.ce.dsg.patrol.platform.bot.RTSGameMainPeer

3) run it.unipr.ce.dsg.patrol.platform.bot.RTSGameMainBot

4) run it.unipr.ce.dsg.patrol.platform.bot.RTSGameMainPeer2

5) run it.unipr.ce.dsg.patrol.platform.bot.RTSMainBot2
